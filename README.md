# Netflix-GPT

- Create React App (npx create-react-app netflix-gpt)
- Add Tailwindcss 
    - npm i -d tailwindcss
    - npx tailwindcss init
    - Configure your template paths (tailwind.config.js)
    - Add the Tailwind directives to your CSS (index.css)
- Routing
- Header
- Login form 
    - Sign In form
    - Sign Up form
    - Form validation
